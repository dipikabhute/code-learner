# **Introduction** 🌐

- ### JavaScript is one of the most popular and powerful programming languages used in web development. It is mainly used to make websites interactive, dynamic, and user-friendly.
- ### Before JavaScript, websites were mostly static, meaning users could only read information. After JavaScript was introduced, websites became interactive and responsive.

- ### Today, JavaScript is used almost everywhere:

## Websites 🌍

### Mobile Applications 📱

### Games 🎮

### Web Servers 🖥️

### APIs 🔗

# **Definition of JavaScript** 📝

- ### JavaScript is a high-level, interpreted scripting language used to create interactive and dynamic webpages.

---
---
## It allows developers to:

- ### Respond to user actions

- ### Change webpage content dynamically

- ### Create animations

- ### Validate forms

- ### Build complete web applications

# **Basic Structure of JavaScript** 💻

<!DOCTYPE html>
<html>
<head>
    <title>JavaScript Example</title>
</head>
<body>

<h1 id="demo">Hello</h1>

<button onclick="changeText()">Click Here</button>

<script>
function changeText() {
    document.getElementById("demo").innerHTML = "Welcome to JavaScript";
}
</script>

</body>
</html>

---

## Explanation of Code 🔍

|Code	              |Meaning|

|<script>	        |Used to write JavaScript|

|function	         |Defines a function|

|onclick	          |   Event that works on click|

|getElementById()	 |Selects HTML element|

|innerHTML	         |Changes webpage content|
