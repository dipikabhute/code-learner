# 1. Organizing Python Codes Using Functions

## What is a Function?

### A function is a block of reusable code that performs a specific task.

### Instead of writing the same code repeatedly, we create a function and call it whenever needed.

## Syntax

## def function_name(parameters):
##     statements
##     return value

## Example

## def greet():
##     print("Welcome to Python")

## greet()

## Output


## Welcome to Python

---

## Components of a Function

## |Component 	|Description                        |

|def	        |Keyword used to define a function  |

|Function Name	|Name given to function             |

|Parameters	    |Input values                       |

|Body	        |Statements executed                |

|Return	        |Sends value back                   |

5. Powerful Lambda Function in Python
What is Lambda Function?

A Lambda Function is an anonymous (nameless) function used for short operations.

Syntax
lambda arguments : expression
Example
square=lambda x:x*x

print(square(5))

Output

25
Lambda with Two Arguments
add=lambda a,b:a+b

print(add(10,5))

Output

15
Lambda with map()
numbers=[1,2,3,4]

result=list(map(lambda x:x*2,numbers))

print(result)

Output

[2,4,6,8]
Lambda with filter()
numbers=[10,15,20,25,30]

result=list(filter(lambda x:x%2==0,numbers))

print(result)

Output

[10,20,30]
Lambda with reduce()
from functools import reduce

result=reduce(lambda a,b:a+b,[1,2,3,4])

print(result)

Output

10
Advantages of Lambda

✔ Short code

✔ No need for function name

✔ Easy with map()

✔ Easy with filter()

✔ Used in sorting

Limitations

✖ Single expression only

✖ Difficult to debug

✖ Not suitable for large programs

Create small, focused functions.
Use meaningful function and module names.
Keep related code together in modules.
Organize large projects into packages.
Reuse code instead of rewriting it.
Add comments and documentation (docstrings) for functions.
Use virtual environments when working with external packages.

Summary
Functions divide a program into reusable blocks of code.
Modules organize related functions, variables, and classes into separate Python files.
Packages group multiple modules into a directory structure for large applications.
Lambda functions provide concise anonymous functions for simple operations and are commonly used with map(), filter(), and reduce().
External packages (such as NumPy, Pandas, and Requests) extend Python's capabilities and are installed using pip.
Using functions, modules, and packages together results in clean, maintainable, reusable, and scalable Python applications.
