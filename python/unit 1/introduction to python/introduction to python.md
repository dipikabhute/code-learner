# 📌  Understanding Python Blocks

## ⭐ What is a Python Block?

A block in Python is a group of statements that are executed together.

👉 Python uses indentation (spaces or tabs) instead of braces {}.

## 📍 Types of Blocks in Python

### 1️⃣ Conditional Block (if-else)

if 10 > 5:
    print("Greater")

✔ Indentation defines the block

### 2️⃣ Loop Block

for i in range(3):
    print(i)

✔ All indented lines are part of loop block

### 3️⃣ Function Block

def greet():
    print("Hello")

✔ Function body is a block

## ⚠️ Important Rule

👉 Incorrect indentation leads to IndentationError

# 📌 Python Basic Operators

## ⭐ What are Operators?

Operators are symbols used to perform operations.

## 📍 Types of Operators

1️⃣ Arithmetic Operators

+  -  *  /  %  //

## ✔ Example:

a = 10
b = 5
print(a + b)

## 2️⃣ Comparison Operators

==  !=  >  <  >=  <=

## ✔ Example:

print(10 > 5) 

 True

## 3️⃣ Logical Operators

and, or, not

## ✔ Example:

print(True and False)

## 4️⃣ Assignment Operators

=  +=  -=  *=  /=

## ✔ Example:

x = 10
x += 5

## 5️⃣ Membership Operators

in, not in

## ✔ Example:

print("a" in "apple")

# 📌 Weakness of Python ⚠️

❌ 1. Slow Execution

Python is slower than C/C++ because it is interpreted.

❌ 2. Memory Consumption

Uses more memory compared to other languages.

❌ 3. Not Ideal for Mobile Apps

Rarely used in mobile development.

❌ 4. Runtime Errors

Errors appear during execution.

❌ 5. Weak in Game Development

Not preferred for high-performance games.

## 🏁 Conclusion

Python is a simple, powerful, and beginner-friendly language used in many modern technologies like AI, Data Science, and Web Development. It is easy to learn but has some performance limitations.
